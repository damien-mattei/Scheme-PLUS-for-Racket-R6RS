Scheme+ for Racket R6RS


Install as a package with the package manager.

Use in an R6RS module:

```(import Scheme+R6RS)```

Designed to use with [SRFI 105 Curly Infix reader for Racket](https://github.com/damien-mattei/SRFI-105-for-Racket).
